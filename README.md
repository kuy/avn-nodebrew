# avn-nodebrew

[![npm version](https://badge.fury.io/js/avn-nodebrew.svg)](http://badge.fury.io/js/avn-nodebrew)

[`nodebrew`](https://github.com/hokaccha/nodebrew) plugin for [`avn`](https://github.com/wbyoung/avn).

## Installation

    $ npm install -g avn-nodebrew
    $ avn setup
